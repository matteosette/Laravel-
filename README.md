# Projet restaurant

Projet de site d'un restaurant construit avec laravel 9.

## Prérequis

- PHP 8.1
- MatiaDB ou MySQL
- composer 2.0.9
- NodeJS 18.12.1 et npm 8.19.2

## installation 

Dans un treminal, lancez : 
```bash
php artisan serve
````

Dans un autre terminal, lancez :

```bash
nmp run dev
```
